Unit 3: Unified Modelling Language (UML)

This unit provided comprehensive coverage of the Unified Modelling Language as a standardised toolset for software system representation. The content explored common UML syntax and examined major UML models used throughout the object-oriented analysis, design and development process. It included practical experience with UML model development using open-source tools.

Required Reading

  • Philips, D. (2018) Python 3 Object-Oriented Programming, Chapter 1
  • Rumbaugh, J., Jacobson, I. & Booch, G. (2004) The Unified Modeling Language Reference Manual, Chapter 3
  • Lange, C. F. J., Chaudron, M. R. V. & Muskens, J. (2006) In practice: UML software architecture and design description
  • Grady, B., Robert, A., Maksimchuk, M., Engle, W., Young, B., Connallen, J. & Houston, K. (2008) Object-oriented analysis and design with applications
  • IBM (2021) UML Models and Diagrams
