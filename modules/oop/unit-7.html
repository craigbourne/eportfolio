Unit 7: Debugging / Error Handling, Data Structures and Data Search

This unit covered debugging concepts, error handling and data structures. The content examined different types of errors, tools for code quality improvement and various data structures and search techniques for processing stored values.

  • Zhou, Z. Q. & Sun, L. (2019) Metamorphic testing of driverless cars
  • Rachum, A. & Kothari, S. (2020) Pydocstyle's documentation
