Unit 1: An Introduction to Python Programming and the OO Programming Paradigm

Module overview

This module introduced the fundamental principles of object-oriented programming and their implementation in Python. The unit covered the evolution of programming languages towards object orientation and examined both the challenges and benefits of object-oriented approaches. Core content included abstract classes, inheritance, encapsulation and polymorphism, alongside basic Python programme structures.

Required Reading

  • Reitz, K. (2016) The Hitchhiker's Guide to Python: Best Practices for Development, Chapter 3
  • Joque, J. (2016) The Invention of the Object: Object Orientation and the Philosophical Development of Programming Languages
  • Seth, A., Aggarwal, H. & Singla, A. (2011) Evolution of Technology through Procedural, Object Oriented and Component Based to Service Oriented
  • Padhy, N., Satapathy, S. & Singh, R.P. (2018) State-of-the-Art Object-Oriented Metrics and Its Reusability: A Decade Review
  • W3C Schools (n.d) Python Data Types
  • Van Rossum, G., Warsaw, B. & Coghlan, N. (2001) PEP 8 -- Style Guide for Python Code
