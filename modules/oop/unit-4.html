Unit 4: Applying a UML Model to a Program Implementation: UML in Practice

This unit bridged theoretical UML understanding with practical implementation. The content focused on creating class diagrams and implementing corresponding Python programmes, developing skills in translating UML models into functional code using PyCharm.

Required Reading

  • Rumbaugh, J., Jacobson, I. & Booch, G. (2004) The Unified Modeling Language Reference Manual, Chapter 3
  • Glinz, M. (2000) Problems and deficiencies of UML as a requirements specification language
  • Uml-diagrams.org (n.d) The Unified Modelling Language
