Unit 11: Pointers, References, Memory and Design Patterns

This unit explored memory management in Python compared to other programming languages, alongside design patterns and sustainable code development principles. The content examined Python's efficient approach to memory handling without direct pointer usage.

Required Reading

  • Reitz, K. (2016) The Hitchhiker's Guide to Python: Best Practices for Development, Chapter 7
  • Nilsson, E. G. (2009) Design patterns for user interface for mobile applications
  • Zhu, H. S., Lin, C. & Liu, Y. D. (2015) A Programming Model for Sustainable Software
  • Philip, G. (2021) Ten Million Users and Ten Years Later: Python Tutor's Design Guidelines
  • Jones, L. (n.d) Pointers in Python: What's the Point?
  • Podder, S., Burden, A., Singh, S. K. & Maruca, R. (2020) How Green is Your Software?
